# 🛡 Cyber Port Scanner

A Professional Web-Based TCP Port Scanner built using Flask, Python, HTML, CSS and JavaScript.

---

## Features

- Scan any IP Address
- Scan Hostnames
- Scan Custom Port Range
- Detect Open Ports
- Detect Closed Ports
- Common Service Detection
- Dashboard
- Search by Port
- Search by Service
- Export Results to CSV
- Modern Cyber Theme
- Responsive Design

---

## Technologies Used

- Python 3
- Flask
- Socket Programming
- HTML5
- CSS3
- JavaScript

## Installation

Clone the project

```
git clone <repository>
```

Go inside project

```
cd CyberPortScanner
```

Create virtual environment

```
python -m venv venv
```

Activate

Windows

```
venv\Scripts\activate
```

Linux

```
source venv/bin/activate
```

Install packages

```
pip install -r requirements.txt
```

Run

```
python app.py
```

Open Browser

```
http://127.0.0.1:5000
```
